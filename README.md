# AIChatbot-with-langchain-agents
This chatbot allows you to talk to your data. Your data must be in a database (i.e. SQLite database). The chatbot can give you the answer as a line chart, bar chart or table.

## Instructions
1. Clone the repo
2. Use requirements.txt to install required libraries
3. Amend config.py with your database path and connection
4. Run main.py
